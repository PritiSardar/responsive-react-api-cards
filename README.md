# Responsive React API Cards

A responsive React application that fetches data from an API and displays it in reusable card components.

## Features

- Fetch data from API
- Display posts in cards
- Responsive grid layout
- Reusable React components
- Tailwind CSS styling

## Tech Stack

- React
- JavaScript
- Tailwind CSS
- Vite

## Concepts Practiced

- useState
- useEffect
- API Fetching
- Props
- Components
- Responsive Design
